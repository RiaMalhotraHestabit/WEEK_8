## Model Files Notice

Due to GitHub file size limitations, large model weight files are **not included in this repository**.

The following files were generated during the quantization experiment but are excluded:

- `model-fp16/`
- `quantized/model-int8/model.safetensors`
- `quantized/model-int4/model.safetensors`
- `quantized/model.gguf`

These files are large (hundreds of MB to multiple GB) and exceed GitHub’s recommended repository size limits.

## DAY-5

## How to Reproduce the Models

The models can be regenerated by running the provided quantization workflow:

1. Load the base model  
2. Apply **INT8** and **INT4** quantization using `bitsandbytes`  
3. Convert the FP16 model to **GGUF format using llama.cpp**

Refer to the notebook/ folder in this repository for the exact commands.

# Local LLM API

This project builds a production-style API for a quantized LLM.

## Features

- FastAPI based LLM server
- Quantized GGUF model
- Text generation endpoint
- Chat endpoint with memory
- Temperature / Top-k / Top-p controls
- Request ID logging

## Endpoints

POST /generate  
POST /chat

## Run

uvicorn deploy.app:app --reload
